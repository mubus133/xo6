A responsive, accessible, and dynamic landing site for Sx06 Luxe built with HTML, CSS, JavaScript, and Bootstrap 5.

- ⭐ Reviews carousel
- 🔗 Linktree-style social hub
- 🎬 Micro-interactions and animations
- 📊 SEO optimized
